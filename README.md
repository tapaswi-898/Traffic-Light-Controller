## Traffic Light Controller 


This project implements a traffic light controller using a Mealy Finite State Machine (FSM) in Verilog HDL. The controller cycles through Red, Yellow, and Green states based on predefined timer values and includes enable and reset functionality.

## Features
Mealy FSM-based design
Timer-controlled state transitions
Active-low asynchronous reset
Enable/disable control
Functional testbench for simulation and verification
State Sequence
OFF → RED → YELLOW → GREEN → RED → ...
## Tools Used
Verilog HDL
Xilinx Vivado
